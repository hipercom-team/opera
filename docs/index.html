  OPERA: An open source project for Wireless Sensor Networks

OPERA is an open source project targeting wireless sensor networks (WSNs). OPERA is destined mainly to the community of developers who work either on simulations or on embedded software for sensor motes. OPERA provides also interesting features for researchers; OPERA allows sensor motes to communicate efficiently while saving energy. It provides a routing protocol coupled with a slot allocation protocol.

OPERA is developed in the ANSI C language and does not depend on a specific underlying OS. This software is an open source under LGPLv3 licence and is available for developers at INRIAGForge (as a git repository).

OPERA is developed in Hipercom team of the Inria institute in the context of the OCARI project (Optimization of Communication for Ad hoc Reliable Industrial network). OPERA belongs to the network layer of the OCARI protocol stack. Embedded within all OCARI protocols in a testbed of sensors, OPERA functionning has been viewed in many open demonstrations.

 OPERA: An efficient tool for energy efficiency in Wireless Sensor Networks

OPERA stands for "OPtimized Energy efficient Routing and node Activity scheduling for wireless sensor networks". As this acronym indicates, this open source software is designed and developed to ensure routing and energy efficiency in Wireless Sensor Networks (WSNs). OPERA provides an adaptive multi-hop routing supporting micro-mobility and enables efficient channel capacity usage. Indeed, OPERA is mainly composed of two modules:

  • Optimized node Activity Scheduling performed by OSERENA. OSERENA is Acronym of "Optimized SchEduling RoutEr node Activity", OSERENA is an activity scheduling mechanism based on node graph coloring. OSERENA helps sensor nodes to save energy because many reasons. First, it avoids collisions and ensures an efficiently use of the bandwidth. Second, based on the activity scheduling provided by OSERENA, wireless sensors can save energy by turning off their radio whenever they are neither transmitting nor receiving.

  • Energy Efficient Routing performed by EOLSR. EOLSR is a proactive energy efficient routing strategy supporting nomadism. EOLSR is based on the known routing protocol OLSR but it is adapted to the context of WSNs: EOLSR is mainly oriented to a typical communication paradigm in WSNs which is data collection and these routes are determined based on an energy criteria.
